1
הורד את האפליקציה SMS Backup & Restore מ-Google Play
SMS Backup & Restore — Carbonite
2
פתח את האפליקציה ← לחץ על "גיבוי" (Backup Now)
סמן רק SMS (לא שיחות) ← לחץ Back Up Now
3
הקובץ נשמר בנתיב /storage/SMS_Backup/ בשם sms-YYYYMMDD.xml או .html
4
העבר את הקובץ למחשב דרך USB / Google Drive / WhatsApp לעצמך
5
לחץ "טען קובץ" כאן ← בחר את הקובץ ← הרה הקופונים מחפש הכל 🎉
1
ב-iPhone אין גיבוי SMS ישיר. הדרך הקלה: iMazing
iMazing — חינמי ל-Export מורידים ל-Mac / Windows
2
חבר iPhone למחשב ← פתח iMazing ← בחר Messages ← Export to CSV
3
שמור כ-CSV ← גרור לכאן — האפליקציה קוראת גם CSV
4
חלופה חינמית: אפשר לגבות ל-iCloud ← הורד ב-Mac דרך Messages.app ← ייצוא ידני

5
לחץ "טען קובץ" כאן ← בחר את ה-CSV ← מצא את הקופונים שלך 🎉
